# 新中式家具产品图册 - 设计风格指南

## 品牌定位

**应用定位**：新中式家具产品图册，展示融合传统东方韵味与现代设计美学的高端家具产品。

**设计风格**：雅致内敛、简约大气、传统与现代交融。追求"少即是多"的东方美学，强调留白与意境。

**目标用户**：追求品质生活、青睐传统文化与现代设计结合的中高端消费者、室内设计师、家具采购商。

## 气质与意象

**核心意象**：
- 徽派建筑的黑白灰对比，马头墙的线条之美
- 宋代文人的清雅书房，宣纸上的墨迹晕染
- 明式家具的线条流畅，榫卯结构的精妙之韵
- 晨光透过格栅窗棂洒在木质桌案，空气中飘着檀香

**关键词**：雅致、简约、留白、意境、线条、自然、沉稳、温润

## 视觉策略

**摄影/图像方向**：
- 产品摄影采用自然光，强调木纹质感与细节
- 背景保持纯净，使用浅灰/米白背景突出产品
- 多用45°侧面视角，展现家具线条与结构之美
- 适当加入中式元素点缀（茶器、花瓶、书画）营造意境

**图形语言**：
- 圆角矩形卡片（圆角较小，保持方正感）
- 线条分隔而非大面积色块分隔
- 传统纹样适度使用（回纹、云纹）作为点缀

## 配色方案

| 色名 | Tailwind 类名 | 意象来源 |
| --- | --- | --- |
| 主色-檀木深 | `text-amber-900`, `bg-amber-900` | 红木、檀木的深沉温润 |
| 辅色-胡桃木 | `text-amber-800`, `bg-amber-800` | 胡桃木的自然暖调 |
| 中性-墨灰 | `text-gray-700`, `bg-gray-700` | 徽派建筑的青瓦墨色 |
| 中性-宣纸白 | `text-gray-50`, `bg-gray-50` | 宣纸的纯净留白 |
| 点缀-朱红 | `text-red-600`, `bg-red-600` | 传统印章、中式点缀色 |
| 点缀-青瓷 | `text-teal-600`, `bg-teal-600` | 青瓷釉色的清雅 |

**应用原则**：
- 页面背景以 `bg-gray-50`（宣纸白）为主
- 文字主色使用 `text-gray-700`（墨灰）
- 分类标签、重要信息使用檀木深色系
- 朱红、青瓷仅在关键交互/强调时使用，点到即止

## 字体排版

**推荐字体**：
- 中文标题：思源宋体（Noto Serif SC）- 体现传统韵味
- 中文正文：思源黑体（Noto Sans SC）- 保持现代阅读舒适度
- 数字/英文：Geometric Sans-serif - 现代简洁

**排版节奏**：
- 大标题：24px（text-2xl），行高 1.4，字间距适度放宽
- 分类标题：18px（text-lg），行高 1.5
- 正文描述：14px（text-sm），行高 1.6
- 价格/参数：16px（text-base），数字略微放大

## 间距系统

| 场景 | Tailwind 类名 | 说明 |
| --- | --- | --- |
| 页面边距 | `px-4`（16px） | 内容区域与屏幕边缘的距离 |
| 卡片内边距 | `p-4`（16px） | 产品卡片内容边距 |
| 卡片间距 | `gap-4`（16px） | 产品列表项间距 |
| 分类标签间距 | `gap-2`（8px） | 分类标签水平间距 |
| 章节间距 | `mb-6`（24px） | 不同内容区块间的距离 |

## 组件使用原则

**组件选型约束**：
- 所有通用 UI 组件优先使用 `@/components/ui/*`，禁止用 `View/Text` 手搓
- 按钮使用 `@/components/ui/button`，风格偏向方圆角、低调配色
- 卡片使用 `@/components/ui/card`，保持简洁留白
- 标签/徽章使用 `@/components/ui/badge`，分类筛选场景
- 分隔线使用 `@/components/ui/separator`，替代手写边框线
- 加载态使用 `@/components/ui/skeleton`，避免灰色方块假 UI

**页面组件规划**：
- 首页：产品卡片（Card）、分类标签（Badge）、搜索框（Input）
- 分类页：分类导航栏（自定义）、产品网格（Card 组合）
- 详情页：大图展示（Image）、参数列表（自定义）、咨询按钮（Button）

## 导航结构

**TabBar 配置**：
- 首页（pages/index/index）- 产品推荐、分类入口
- 分类（pages/category/index）- 分类列表、产品筛选
- 我的（pages/profile/index）- 收藏、浏览历史、关于品牌

**页面跳转规范**：
- TabBar 页面切换使用 `Taro.switchTab()`
- 进入详情页使用 `Taro.navigateTo()`

## 小程序约束

**包体积**：
- 严格控制图片资源，产品图片使用 TOS 对象存储
- TabBar 图标使用本地 PNG（81x81），其余图片走 TOS URL

**性能优化**：
- 产品列表使用虚拟滚动（长列表场景）
- 图片使用懒加载
- 避免复杂的动画效果，保持简洁

## 设计禁忌

**明确"不要做"清单**：
- 不要使用大面积彩色渐变背景
- 不要使用过于圆润的设计元素（保持方正雅致）
- 不要堆砌过多装饰元素（保持留白）
- 不要使用卡通化图标风格
- 不要使用过于鲜艳的配色（保持沉稳内敛）
- 不要在页面内用 View/Text 手搓通用组件（Button/Input/Card 等）
